Запускать нужно из консоли.
Изначально нужно ввести свой ключ и токен TrelloApi в input
Чтобы добавить колонку, нужно вызвать метод "create_column" и передать ему имя колонки в скобках
Если вы хотите перенести задачу, но таких задач несколько, input попросит у вас указать название колонки из которой нужно
перенести (без кавычек), а потом номер повторной задачи в этой колонке (его также можно увидеть в консоли), если такое
задание единственное в колонке, номер = 1
